# titanic
## Titanic Survival Prediction

Меня зовут Акылай, я учусь в 10 классе и начинаю изучать анализ данных и машинное обучение. Этот проект — моё первое практическое задание в этой сфере.

# Цель проекта

Цель проекта — предсказать выживаемость пассажиров Титаника на основе их характеристик. Модель использует такие признаки, как:
пол пассажира
возраст
класс билета
количество членов семьи на борту
порт посадки

**Данные**

Для анализа используются данные из набора train.csv и test.csv. train.csv содержит информацию о пассажирах и известные результаты выживания, а test.csv используется для проверки качества предсказаний модели.


**Методы**

Проект включает следующие шаги:
Предобработка данных – обработка пропусков, кодирование категориальных переменных, создание новых признаков.

Анализ данных – визуализация распределений и взаимосвязей признаков с выживаемостью.

Построение модели – использование алгоритмов машинного обучения для предсказания выживания пассажиров.

Оценка модели – проверка точности предсказаний на тестовой выборке.
