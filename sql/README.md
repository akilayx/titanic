# Реестр SQL-запросов

В этой папке находятся SQL-скрипты, используемые для аналитических запросов и извлечения признаков.

## Список запросов:

- avg_median_purchase.sql — Средняя и медианная сумма покупок на одного пассажира
- total_count.sql — Общее количество пассажиров
- top_spent_passanger.sql — Пассажиры с наибольшими расходами
- top_nulls.sql — Пассажиры с наибольшим количеством пропущенных значений
- top_alcohol_spent.sql — Пассажиры с наибольшими расходами на алкоголь
- survived_no_purchase_stats.sql — Статистика выживаемости пассажиров без покупок
- survival_stats.sql — Общая статистика выживаемости
- purchases_below_0.sql — Некорректные значения покупок (отрицательные)
- nulls_count.sql — Подсчёт пропущенных значений
- no_purchase_passanger.sql — Пассажиры без совершённых покупок
- lifeboat_status_count.sql — Статистика использования спасательных шлюпок
- difference_pclass.sql — Различия в расходах по классам пассажиров
- category_sum.sql — Общая сумма расходов по категориям
- category_spent.sql — Распределение расходов по категориям
- category_avg_spent.sql — Средние расходы по категориям
